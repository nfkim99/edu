1. HelloWorld 를 출력하는 프로그램의 과정을 설명하시오

	과정: 소스코드 -> 컴파일 -> 실행
		           javac   -> java

	1- 소스코드 짜기 .java (인간이 이해할수있는 언어)

	class HelloWorld {
    		public static void main(String[] args){
        			System.out.println("HelloWorld");
    		}
	}
			
	

	2- cmd -> 해당 .java 파일있는 폴더 에가서
	3- javac Hello1.java  <--- // javac 명령어로 컴파일 실행
		Hello1.class 파일 생성됨 <-- //컴퓨터가 알어먹을수있는 언어로 컴파일됨
	4- java Hello1 <-- java 명령어로 컴파일된 Hello1.class를 실행하게된다.
			여기서 .class는 생략가능하다.

//////// 출력결과 /////
HelloWorld




2. 아래의 명령어를 설명하시오(뭐 하는 건가).
-javac
	javac 는 인간이 짜놓은 소스코드 .java 파일을 컴퓨터가 이해할 수 있게 컴파일해주는 명령어이다.
	컴퓨터는 0101010101로 구성된 것만 이해할수 있으므로 컴파일하여 .class파일을 생성한다.
	이 .class파일이 컴퓨터가 이해할수있고, 실행할수가 있다.

-java
	java 명령어는 javac로 인해 컴파일된 .class파일을 실행하는 명령어이다.
	이 명령어를 통해 우리가 짜놓은 소스코드의 출력값이나 프로그램을 실행할수있다.


3. 컴파일이란 무엇인가요?
	컴파일이란, 고급언어(C,C++,JAVA 등으로 인간이 구분하기 쉬운 언어)로 작성된 프로그램을 기계어(컴퓨터 등의 기계가 이해할 수 있는 언어)로 
	번역하는 것을 컴파일이라 한다. 
	이것을 처리하는 프로그램을 컴파일러라고 한다.
	앞서 말한거와 같이 컴퓨터는 0101010101로 구성된 것만 이해할수 있으므로 
	컴파일하여 .class파일을 생성한다.
